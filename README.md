# GivEnergy for Homey

[![Buy Me a Coffee](https://img.shields.io/badge/buy%20me%20a%20coffee-donate-yellow)](https://buymeacoffee.com/jakio)
[![GitHub Sponsors](https://img.shields.io/github/sponsors/jak?label=sponsor)](https://github.com/sponsors/jak)

Monitor and control your GivEnergy solar inverter, battery, and grid meter locally from [Homey](https://homey.app).

This is an unofficial community app and is not affiliated with, endorsed by, or related to GivEnergy Ltd. Only GivEnergy solar inverters and batteries are supported — EV chargers are not supported.

## Features

- Real-time solar, battery, and grid power monitoring
- Battery and grid energy tracking in Homey's Energy tab
- Flow cards for inverter mode, charge/discharge schedules, and rates
- Direct local connection via Modbus TCP — no cloud dependency
